# HTML-CSS1
HTML/CSSの勉強
架空のカフェのホームページ作成します

このホームページの作成を通して、HTML/CSSの基礎を理解し、自分で扱えることを目的とします。

スケジュール

1/23ー1/24：　indexを作成

シングルカラムであるindexページを作成しました。

まず、WEBページの情報を記述する「head」の作成を通して、タイトル名の表記方法や文字コードの指定、ress.cssやオリジナルのCSSの指定方法などを学びました。

次に、WEBページ上部に表示させる「header」の作成しました。ロゴの配置を通して画像の挿入方法や幅や余白の指定方法を、ナビゲーションメニューの作成を通して文字の装飾やレイアウト、クラス作成を学びました。中でもポイントは、ナビゲーションメニューを「display:flex;」を用いて横並びにしたことです。

次に、ページの上部の中心部分に配置する「キャッチコピー」とその背景を彩る「カバー画像」を作成しました。
