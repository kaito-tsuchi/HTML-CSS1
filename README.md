# HTML-CSS1
HTML/CSSの勉強
架空のカフェのホームページ作成します

このホームページの作成を通して、HTML/CSSの基礎を理解し、自分で扱えるようになることを目的とします。

スケジュール

1/23ー1/24：　indexを作成

シングルカラムであるindexページを作成しました。

この作業では、「head部分」→「ロゴ、ナビゲーションメニュー」→「コンテンツ部分」→「ファビコン追加」の順で作成した。

まず、WEBページの情報を記述する「head」の作成を通して、タイトル名の表記方法や文字コードの指定、ress.cssやオリジナルのCSSの指定方法などを学びました。

次に、WEBページ上部に表示させる「header」の作成しました。ロゴの配置を通して、画像の挿入方法や幅や余白の指定方法を学びました。ナビゲーションメニューの作成を通して、文字の装飾やレイアウト、クラス作成を学びました。中でもポイントは、ナビゲーションメニューを「display:flex;」を用いて横並びにしたことです。

次に、ページの上部の中心部分に配置する「キャッチコピー」とその背景を彩る「カバー画像」を作成しました。キャッチコピーの作成を通して、文字のフォントサイズ指定やレイアウト、装飾を学びました。カバー画像作成では、背景画像の設定方法を学びました。

次に、カフェの概要を説明するコンテンツ部分の作成を行いました。見出しや本文部分の作成方法やボタンの設置方法を学びました。

次に、Webページ下部に表示させる「フッター部分」の作成を行いました。「フッター部分」の作成を通して、一行ごとに1本の選を引く方法やコピーライトの追加方法を学びました。

次に、「レスポンシブ対応」を行いました。まず、レスポンシブ対応とは、デスクトップのパソコンやスマートフォンなどで表示領域の幅が異なるとき、見え方が変わるようにすることです。レスポンシブ対応では、cssにデスクトップ版のスタイルを追加し、見出しやヘッダー、HOME、フッターなどのデスクトップ版の装飾を追加しました。この際、モバイルサイズとデスクトップサイズで、別の画像を用意することで、画面を見やすくしました。

最後に、「ファビコン」の追加を行いました。

1/25ー1/26：　Newsページを作成

2カラムであるNewsページを作成しました。

この作業では、「見出し」→「メインエリア」→「サイドバー」→「レスポンシブ対応」の順で作成しました。

まず、「index.html」をコピーして、「news.html」 を作成し、indexページで学んだことを活かして、サイドバー以外の部分を作成しました。<news.html>の<article>タグの下に<aside>タグを追加し、サイドバーに掲載するコンテンツを書いていくことで「サイドバー」を作成し、2カラムのページを実現しました。この際にカテゴリーやアーカイブのリストを＜ul＞タグで作成することを学びました。

1/27ー1/28：　Menuページの作成

タイル型のレイアウトのMenuページを作成しました。

この作業では、「見出し」→「タイル型レイアウトの設定」→「レスポンシブ対応」の順に作成しました。

まず、「news.html」をコピーして、「menu.html」 を作成し、学んだことを活かして、タイル型レイアウト以外の部分を作成しました。タイル型レイアウトの設定では、画像とテキストは、「item」というクラスの<div>タグで囲み、さらに9つの「item」を「grid」というクラスのついた＜main＞タグで囲むことで、コンテンツの用意をしました。cssの装飾では、親要素である「grid」クラスに対して、「display:grid;」を指定することでタイル型に並ぶようにしました。また、「grid-template-colums」で、値を「1lf,1lf」とすることで、1：1の割合で2つの要素を横に並べる方法を学びました。

1/29ー1/31：　Contactページの作成

この作業では、「見出し」→「地図の表示」→「メールアドレス宛てのボタンの表示」→「Instagramの投稿を挿入」→「Xプラグインを挿入」→「YouTube動画の挿入」の順で作成した。
